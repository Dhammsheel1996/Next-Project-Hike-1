# Web-Scarping-and-Word-Frequeny-Project
Using this script we can analyze the word counts
Functionality
Utilites libraries such as BeautifulSoup and requests to collect data from the specific websites.
Extract the textual contents form the choosen URL
Featch the latest articles titiles through wWebscraping Python Module.
Counts and displays the frequency of ech word found whithin those articles titles
Provide insights into the mose commonly used words in the titles.
Dependencies
BeautifulSoup : For parsing HTML Contents.
Requests : For handling HTTP requests.
Collections : For tally word occurrence.
